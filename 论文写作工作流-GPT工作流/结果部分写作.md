核心：**图或者表格的阐述**

1）目的与范围

- 关键词：目的 / 聚焦 / 本节旨在 / 本节关注 / 本节限定在 / 对比对象 / 任务场景
    
- 句式骨架：本节旨在……，聚焦于……任务中……与……的对比，其他条件保持一致。
    

2）评价指标与判据

- 关键词：评价指标 / 主要指标 / 辅助指标 / 统计口径 / Top-1 Accuracy / Macro-F1 / Precision / Recall / 参数量 / FLOPs
    
- 句式骨架：采用……作为主要指标，并以……作为补充，用于从……角度刻画性能。
    

3）引出结果载体并先给论点

- 关键词：如表/图所示 / 汇总于 / 结果概览 / 核心发现 / 结论是 / 总体趋势 / 主要现象
    
- 句式骨架：相关结果见表/图……。总体上，……呈现……，表明……。
    

4）定性描述

- 关键词：整体趋势 / 相对关系 / 更优 / 不劣 / 更稳定 / 差异主要体现在 / 一致性 / 鲁棒性
    
- 句式骨架：从整体趋势看，……；差异主要体现在……。
    

5）定量论据（逐项指标落到数字）

- 关键词：分别为 / 提升 / 降低 / 绝对提升（pp）/ 相对提升（%）/ 平均 / 标准差 / 波动范围 / 统计显著
    
- 句式骨架：与……相比，……在……指标上由……变为……（提升……pp / ……%）。
    

6）原因解释（机制/约束/误差来源）

- 关键词：可能原因 / 归因 / 机制解释 / 光学卷积 / 频域滤波 / 系统噪声 / 量化 / 对准误差 / 有效带宽 / 信息保真
    
- 句式骨架：上述差异可能源于……，其导致……，进而……。
    

7）升华与承接（意义 + 指向下一节）

- 关键词：说明 / 支持 / 证明 / 体现 / 为后续 / 奠定基础 / 推广到 / 更复杂任务
    
- 句式骨架：综上，结果表明……；该结论为后续在……上的进一步验证提供依据。
